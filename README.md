# My Gantt Project

This is a **Next.js + TypeScript** app demonstrating a Gantt chart using **dhtmlxGantt** with:
- Japanese language support
- Viewport-resizing behavior
- Zoom and density controls

## Features

- Gantt chart embedded in a Next.js page
- Japanese UI text
- Task zoom (Day / Week / Month)
- Row density control (Compact/Normal/Large)

## How to Run

```bash
git clone https://github.com/ekawai/my-gantt.git
cd my-gantt
npm install
npm run dev
